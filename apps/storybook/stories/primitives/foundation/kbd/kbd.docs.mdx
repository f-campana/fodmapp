## When to use

Use Kbd when the interface needs to teach or remind users about an existing
keyboard interaction.

Avoid it as decorative typography or as a substitute for the actual command
implementation.

## Accessibility contract

- the primitive renders semantic `<kbd>` content for keyboard input notation
- grouped shortcuts should still read clearly in surrounding sentence copy
- the group wrapper keeps a stable slot hook when multiple keys are shown

## Validation

What is checked automatically:

- Storybook verifies both single-key and grouped-shortcut rendering

What is checked manually:

- visual pass for inline readability and balanced spacing inside support copy
