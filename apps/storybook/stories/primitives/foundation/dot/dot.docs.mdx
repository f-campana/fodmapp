## When to use

Use Dot for compact state indicators inside dense rows, legends, or support
copy where a full badge would add too much weight.

Avoid it when the meaning would be ambiguous without surrounding text.

## Accessibility contract

- Dot should be paired with visible or assistive text when it carries meaning
- size and color are visual only unless the surrounding content explains them
- the root keeps a stable slot hook for regression checks

## Validation

What is checked automatically:

- Storybook verifies the default rendering and custom accessible label case

What is checked manually:

- visual pass for size clarity and contrast when used inline with text
