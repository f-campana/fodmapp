## When to use

Use Badge for concise state, severity, or review labels that do not respond to
user input.

Use Chip instead when the user must toggle or remove the token.

## Accessibility contract

- Badge is a passive text container, not an interactive control
- variant changes are visual only unless surrounding copy provides the meaning
- the root keeps a stable `data-slot` hook for automation and regression checks

## Validation

What is checked automatically:

- Storybook verifies variant data attributes in a realistic status-card context
- interaction checks cover the stable slot hook and rendered variant marker

What is checked manually:

- visual pass for tone contrast and short-label legibility in compact layouts
