## When to use

Use Label whenever a form control needs visible naming in the same local field
context.

Avoid using placeholder text as the only field name.

## Accessibility contract

- association comes from `htmlFor` and the target control id
- Label provides visible naming only; required, invalid, and disabled semantics
  still come from the field and control
- the root keeps a stable slot hook for automation and styling

## Validation

What is checked automatically:

- Storybook verifies click-to-focus association and the disabled-field case

What is checked manually:

- visual pass for field rhythm, helper-copy alignment, and readable label copy
