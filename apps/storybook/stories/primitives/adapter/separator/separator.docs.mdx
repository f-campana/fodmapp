## When to use

Use decorative separators for simple visual rhythm between related blocks.

Use non-decorative separators only when the split represents real structure
between neighboring regions.

## Accessibility contract

- `decorative` hides separator semantics from assistive technologies
- semantic separators expose the correct orientation when they represent
  structure
- the root keeps a stable slot hook in both modes

## Validation

What is checked automatically:

- Storybook verifies horizontal decorative and vertical semantic cases

What is checked manually:

- visual pass for stroke clarity, inset spacing, and vertical alignment
