## When to use

Use AspectRatio when image or video loading should not shift the surrounding
layout.

Avoid it when the parent container does not have a meaningful width yet or when
the child content should size itself naturally.

## Accessibility contract

- the primitive only reserves layout space; accessible naming still comes from
  the child media content
- the root keeps a stable `data-slot` hook for layout and regression checks

## Validation

What is checked automatically:

- Storybook verifies the reserved box inside the local audit frame
- interaction checks cover the root slot marker and rendered media presence

What is checked manually:

- visual pass for cropping, border radius, and constrained-parent behavior at
  compact widths
